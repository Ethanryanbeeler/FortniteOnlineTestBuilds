# Fortress
Fortress was the internal codename for Fortnite during its early development at Epic Games. Before the game became the battle royale phenomenon it is today, it was a cooperative survival game focused on building structures and fighting off waves of monsters called Husks.

### Key Details
| Detail | Info |
|--------|------|
| Developer | Epic Games |
| Engine | Unreal Engine 4 |
| Genre | Co-op Survival / Tower Defense |
| Original Concept | Build forts and defend against Husks |
| Announced | 2011 (VGA Awards) |
| Released (Early Access) | July 25, 2017 |
